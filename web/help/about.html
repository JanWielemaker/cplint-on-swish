cplint on SWISH is a web application for trying probabilistic logic programming with a Javascript-enabled web browser on any operating system. It was written by Fabrizio Riguzzi, Riccardo Zese and Giuseppe Cota.

SWISH was originally written by Torbjörn Lager as a homage to SWI-Prolog. Jan Wielemaker designed and implemented the present version.

The cplint on SWISH source is available from Github. It requires SWI-Prolog 7 installed from the latest GIT.
